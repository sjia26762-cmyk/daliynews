# 多地开展老旧小区燃气管道改造工程 新增安全防护设施超3万套

**2026年7月17日 12时34分56秒**

---

近日，住建部联合应急管理部启动老旧小区燃气管道改造工程，重点改造2000年前建成的老旧小区燃气管道，更换老化管道、安装安全防护设施，消除燃气安全隐患，上半年全国新增燃气安全防护设施3.2万套。此次改造主要包括更换镀锌钢管为无缝钢管、安装燃气泄漏报警器、紧急切断阀、防雷防静电设施等，同时建立燃气安全监测系统，实现燃气管道运行实时监测。在天津，当地对600个老旧小区进行燃气管道改造，更换老化管道450公里，安装燃气泄漏报警器1.2万个；在西安，改造燃气管道300公里，安装紧急切断阀8000余个，实现燃气泄漏自动切断。截至目前，各地已完成1500个老旧小区燃气管道改造，消除燃气安全隐患4.5万处。相关部门表示，年底前将完成4000个老旧小区燃气管道改造，切实保障居民生命财产安全。
