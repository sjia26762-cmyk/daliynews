# 我国科学家在人工合成淀粉领域取得新突破 合成效率提升3倍

**2026年7月17日 12时34分56秒**

---

近日，中国科学院天津工业生物技术研究所的科学家在人工合成淀粉领域取得新突破，通过优化合成路径、改造关键酶，实现人工合成淀粉效率较此前提升3倍，相关成果发表在国际顶级学术期刊《科学》上。人工合成淀粉是通过二氧化碳、氢气等原料，在催化剂作用下合成淀粉，可替代传统农业种植，缓解粮食安全压力。此次研究中，科学家通过基因编辑技术改造合成酶，优化反应条件，将淀粉合成效率从每升每小时0.5克提升至1.5克，同时降低了能耗和成本。目前，该技术已进入中试阶段，预计2028年可实现工业化生产。据测算，工业化生产后，1立方米生物反应器每年可生产淀粉5吨，相当于5亩玉米地的产量。相关科学家表示，此次突破为解决粮食安全、碳中和等问题提供了新路径，未来将进一步优化技术，推动产业化应用。
