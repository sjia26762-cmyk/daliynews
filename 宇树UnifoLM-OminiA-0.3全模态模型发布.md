# 宇树UnifoLM-OminiA-0.3全模态模型发布

**2026年7月17日 12时34分56秒**

---

7月宇树科技发布UnifoLM-OminiA-0.3，支持全模态交互理解、全程自主运行与稳定执行抗干扰，面向四足与人形通用控制。过去机器人控制栈分“感知模型+运动控制器+任务规划”三层拼凑，该模型尝试把视觉、语音、力觉、关节状态编入同一序列做端到端决策。演示场景含不平地面抗推、噪声指令纠偏、陌生物体抓取。对开发者，宇树开放模型意味着四足从“遥控狗”变“可派活员工”；对行业，全模态控制是具身智能跳过遥操作数据瓶颈的尝试。风险在真实工厂长尾干扰仍可能触发死机，需OTA持续喂数据。
