# 我国首座大型生物质能发电站扩建工程完工 年发电量超3亿千瓦时

**2026年7月17日 12时34分56秒**

---

近日，我国首座大型生物质能发电站——山东单县生物质能发电站扩建工程完工，扩建后总装机容量达30万千瓦，年发电量超3亿千瓦时，可满足15万户家庭一年的用电需求，标志着我国生物质能发电技术取得新进展。该发电站以农作物秸秆、林业废弃物等为原料，通过燃烧发电，实现了农林废弃物资源化利用，每年可消耗农林废弃物250万吨，减少二氧化碳排放28万吨。扩建工程新增装机容量15万千瓦，采用先进的生物质燃烧技术和环保设备，发电效率较此前提升20%，污染物排放远低于国家排放标准。据了解，该发电站还为当地提供了200余个就业岗位，带动了周边农林废弃物收集、运输等产业发展。相关企业表示，未来将在全国农林主产区推广生物质能发电站建设，推动我国可再生能源产业发展。
